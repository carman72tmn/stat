# Payroll Admin Panel

Полноценный проект для расчёта заработной платы сотрудников с административной панелью, регистрацией пользователей, ролями и правами доступа.

## Что реализовано

- Регистрация и авторизация пользователей.
- Полноценная админ-панель Django для управления пользователями, профилями и расчётами.
- Назначение прав доступа через роли (группы): `HR`, `Accountant`, `Manager`, `Employee`.
- Расширенный калькулятор зарплаты:
	- базовая ставка;
	- отработанные часы;
	- переработки с коэффициентом;
	- бонусы и надбавки;
	- удержания;
	- налоговая и страховая ставки;
	- автоматический расчёт начислений и суммы к выплате.

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py init_roles
python manage.py runserver
```

Откройте:

- `http://127.0.0.1:8000/` — пользовательская панель и калькулятор.
- `http://127.0.0.1:8000/admin/` — административная панель.

## Права и роли

Команда `python manage.py init_roles` создаёт группы и назначает права:

- `HR` — управление пользователями и просмотр всех расчётов.
- `Accountant` — выполнение расчётов для всех сотрудников и изменение расчётов.
- `Manager` — просмотр всех расчётов.
- `Employee` — создание и просмотр собственных расчётов.

Назначение ролей пользователям выполняется в админке:

`Admin -> Users -> <user> -> Groups`
